# Beneficios de las L1

Crear una blockchain personalizada como una L1 en Avalanche ofrece ventajas que van mucho más allá de simplemente tener tu propia red. Estas ventajas permiten diseñar un entorno totalmente optimizado para tu caso de uso, sin las limitaciones que imponen otras cadenas compartidas o soluciones L2.

## 1. Token de gas personalizado

En una L1 de Avalanche, tú decides qué token será utilizado para pagar las tarifas de gas. Esto significa que puedes alinear directamente la economía de tu red con tu propio token, incentivando su uso y creando modelos económicos a medida.

## 2. Alto rendimiento

Cada L1 maneja sus propias transacciones y no comparte espacio de bloque con otras cadenas. Esto evita la congestión y mantiene las tarifas bajas, incluso si la red principal de Avalanche o cualquier otra L1 dentro del ecosistema está saturada o bajo alta demanda.

## 3. Control de acceso estricto

Puedes definir quien tiene acceso para desplegar contratos inteligentes y quién puede interactuar con ellos. Esto es esencial para casos de uso institucionales, gubernamentales o regulados, donde el cumplimiento normativo requiere un control granular de los participantes.

## 4. EVM personalizada o VM diferente

Si Solidity y la EVM estándar no son suficientes o no son económicamente óptimas para tu caso de uso, puedes personalizar la EVM o incluso implementar una maquina virtual completamente distinta (por ejemplo, una VM optimizada para juegos o para procesamiento de datos). Esto abre la puerta a nuevas arquitecturas y funcionalidades únicas.

## 5. Escalabilidad garantizada

Al ser una chain independiente, tu L1 puede crecer y escalar sin competir con otras aplicaciones o redes por recursos. Además puedes ajustar parámetros como el tamaño de bloque, el tiempo de confirmación y el mecanismo de consenso para lograr el rendimiento exacto que necesitas.
